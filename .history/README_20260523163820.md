# Allo Health Engineering Take-Home

## Setup

```bash
npm install
npx prisma generate
npx prisma migrate dev
npm run seed
npm run dev
```

## Environment Variables

Create `.env`:

```env
DATABASE_URL="postgresql://..."
```

## Concurrency Strategy

Reservations are protected using PostgreSQL row-level locking (`FOR UPDATE`) inside a Prisma transaction.

This guarantees that if two users attempt to reserve the final unit simultaneously:
- one succeeds
- one receives HTTP 409

This prevents overselling.

## Expiry Strategy

Expired reservations are lazily cleaned up before reservation operations.

A production deployment can additionally use a Vercel Cron Job every minute.
